💡 CUSTOMIZATION TIPS
═══════════════════════════════════════════════════════════════════════════

ADD NEW NAVIGATION ITEM:
1. Add to appropriate role dropdown
2. Use consistent icon and styling
3. Update URL routing
4. Test on all devices

ADD NEW ROLE:
1. Add role detection logic
2. Create role-specific dropdown
3. Add role colors and icons
4. Update user dropdown

CHANGE COLORS:
1. Update CSS variables
2. Modify role color logic
3. Update badge classes
4. Test accessibility

ADD BREADCRUMBS:
1. Pass $breadcrumbs array from controller
2. Structure: ['title' => 'Page', 'url' => 'path']
3. Last item has no URL (active page)
4. Test navigation flow

📍 QUICK DEBUGGING
═══════════════════════════════════════════════════════════════════════════

If navigation shows wrong role:
1. Check get_user_role() function
2. Verify session data
3. Check role detection logic
4. Clear browser cache

If dropdowns don't work:
1. Check Bootstrap JS loaded
2. Verify dropdown IDs match
3. Check JavaScript console
4. Test on different browsers

If styling issues:
1. Check CSS variables
2. Verify Bootstrap classes
3. Check responsive breakpoints
4. Validate HTML structure

If breadcrumbs missing:
1. Pass $breadcrumbs from controller
2. Check array structure
3. Verify template logic
4. Test with sample data
